# Car Rental System (Java Console Application)

A simple **Car Rental Management System** built using **Java**.  
This program allows customers to rent and return cars, view available vehicles, and manage rental details through a console-based menu.

## Features

- **Rent a Car**
 - Displays all available cars.
 - Takes customer details.
 - Allows choosing a car by ID.
 - Calculates rental price based on number of days.
 - Confirms and books the rental.

- **Return a Car**
 - Customers can return rented cars.
 - Updates availability.
 - Displays return confirmation.

- **View Cars**
 - Shows all available and rented cars.

- **Customer Management**
 - Each customer gets a unique Customer ID.
 - Stores customer rental details.

- **Menu-driven Program**
 - Runs in a loop until the user exits (Option 8).
 - Prevents invalid inputs with proper validations.
